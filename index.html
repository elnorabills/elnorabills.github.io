Some Things I've Built

Threaddit

This is a full stack application built with Python, Flask, WTForms, SQLAlchemy, JavaScript, React, Redux, HTML and CSS. Implemented with the ability for users to create, read, update, and delete (CRUD) content by building a flask application on the backend with refined routes to handle each request sent from the frontend. Developed the frontend to create a clean user interface through the use of React, which enables a modular structure of reusable components, and Redux, which holds the global state of the application in a single store. Features to add and improve include a search bar, a sorting functionality to filter through various sub-threaddits, and the ability to add pictures to a post through the use of AWS (Amazon Web Services).

Stack Overflow

This application was built as a team with three other developers. We divided the work using organizational tools such as Asana to conquer every task necessary to complete a fully functional application in under two weeks. Features include the ability to post questions, post answers to a questions, post comments to answers, and upvote or downvote answers. Future features include AWS (Amazon Web Services), a rich text editor for posting answers on questions, a search bar, the ability to upvote or downvote questions as well as answers, and the ability to edit comments.

SoundCloud

This is my first solo full stack project and includes technologies such as JavaScript, Express, Sequelize, Node.js, React, Redux, HTML, and CSS. Users are able to create, read, update, and delete both songs and albums. This project uses Sequelize as an ORM (Object Relational Mapper) in order to handle database records with relational databases such as SQLite3 (used in development) and PostgreSQL (used in production). Future features to be implemented include the ability to comment on songs and albums, create playlists, and utilize AWS (Amazon Web Services) in order to add a cover photo on songs, albums, and playlists.
